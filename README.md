# Tienda PHP

Tienda online desarrollada en **PHP** usando **Apache/XAMPP**.  
Esta aplicación permite ver y probar la tienda directamente desde los archivos subidos al repositorio.  

## Archivos principales

- `index.php` – Página principal de la tienda.  
- `hola.php` – Archivo de prueba o ejemplo.  
- `db.php` – Conexión a la base de datos.  
- `style.css` – Estilos de la tienda.  
- `test.php` – Archivo de test.  
- `imágenes/` – Carpeta con todas las imágenes de la tienda.  
- `js/` – Carpeta con scripts JavaScript.  
- `tienda web/` – Otros archivos relacionados con la tienda.

Abrir la carpeta en XAMPP o tu servidor Apache local.
